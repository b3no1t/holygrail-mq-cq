# Lexique des Termes

**Container Query** : Technique CSS permettant d'appliquer des styles en fonction de la taille d'un conteneur parent plutôt que de la fenêtre du navigateur (viewport).
**Container** : Élément HTML défini comme point de référence pour les requêtes de conteneur via la propriété container-type.
**Containment Context** : Contexte créé par un élément container qui isole ses dimensions des éléments extérieurs.
**Container Query Length Units** : Unités relatives à la taille du conteneur (``cqw``, ``cqh``, ``cqi``, ``cqb``, ``cqmin``, ``cqmax``).
**Media Query** : Technique traditionnelle CSS basée sur les dimensions du viewport (écran).
**Viewport** : Zone d'affichage visible du navigateur.
**Holy Grail Layout** : Pattern de mise en page classique avec header, footer, contenu principal central et deux sidebars latérales.
**Flexbox** : Système de mise en page CSS unidimensionnel pour distribuer l'espace et aligner les éléments.
**Responsive Design** : Approche de conception adaptant l'affichage aux différentes tailles d'écran.
**Breakpoint** : Point de rupture définissant un changement de mise en page.
**Intrinsic Sizing** : Dimensionnement basé sur le contenu interne d'un élément.
